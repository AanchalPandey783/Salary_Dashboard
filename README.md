# Data Jobs Salary Dashboard

**INTRODUCTION :**
This dashboard provides insights into salaries across various data-related roles, helping users explore compensation trends, job locations, and in-demand skills.

<img width="814" height="322" alt="image" src="https://github.com/user-attachments/assets/c2e9fff2-ca1b-4122-b79b-b822dc60c313" />


Built using a real-world 2023 data jobs dataset, the dashboard analyzes:
👨‍💼 Job Titles
💰 Salaries
📍 Locations
🛠️ Required Skills
Excel Skills Applied
📉 Data Visualization (Charts)
🧮 Formulas & Functions
✅ Data Validation


**Salary by Job Title**
📊 Excel Features: Used a horizontal bar chart with formatted salary labels.
🎨 Design Choice: Sorted salaries in descending order for easy comparison.
👀 Visualization Benefit: Makes high- and low-paying roles instantly identifiable.
💡 Key Insight: Senior-level and engineering roles generally offer higher salaries than analyst positions.

**Global Salary Distribution**
🌍 Excel Features: Created a map chart to display median salaries by country.
🎨 Design Choice: Applied color gradients to highlight salary differences across regions.
📈 Data Representation: Visualizes salary trends on a global scale.
💡 Key Insight: Reveals geographic variations and identifies higher- and lower-paying markets.

**Median Salary Analysis**
🧮 Excel Features: Used a dynamic array formula combining MEDIAN() and conditional logic.
🎯 Purpose: Calculates median salaries based on selected job title, country, and work type.
📊 Benefit: Provides customized salary insights for different criteria.
💡 Outcome: Enables targeted salary comparisons across roles and locations.
